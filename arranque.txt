Procedimiento para iniciar:
    1.-Clonar repositorio via github
    2.-Activar entorno virtual desde la terminal con el siguiente comando (en windows):
        CRUD-env\Scripts\activate
    4.-Moverse a la carpeta CRUD desde el terminal
    5.-Utilizar el comando para iniciar:
        python manage.py runserver
